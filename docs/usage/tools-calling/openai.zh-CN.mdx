## GPT-4 Turbo

### 简单调用指令：天气查询

测试指令：指令 ①

GPT-4 Turbo 在调用 Tool Calling 时并没有像 GPT-3.5 Turbo 一样回复「好的」，且经过多次测试始终一样，因此在这一条复合指令的跟随中反而不如 GPT-3.5 Turbo，但剩余两项能力均不错。

当然，也有可能是因为 GPT-4 Turbo 的模型更加有“自主意识”，认为不需要输出这一句“好的”。
